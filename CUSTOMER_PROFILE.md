```  
        https://api.hojrenama.com/web/user/edit
```  

 ```  
POST
 ```
*  **URL Params**

```
@username
@token
@x
@y
@name
@family
$phone
$email
$state
$city
$hood
$address
$age
$gender
$telegram
$twitter
$linkedin
$facebook
$instagram
$googleplus
$website
$avatar
```

    ```
"message": "updated successfully",

}
    ```
